## See

> GithubPages

`yarn add gh-pages`, then in "package.json" add scripts:

```js
"homepage": "https://ndrean.github.io/modals-react"
"scripts:{
  "predeploy": "yarn build",
  "deploy": "gh-pages -d build"
}
```

and visit https://ndrean.github.com/ndrean/modals-react
